= Assign tags to clusters
:toc: macro
:toclevels: 1
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Beginning with ONTAP 9.14.1, you can use System Manager to assign tags to clusters to identify objects that belong to a category, for example, a project or a cost center.

.About this task
Tags can be added when you create a cluster, or they can be added later.

include::_include/tags-clusters-volumes-sm.adoc[]

.Steps
To assign a tag to a cluster, you need to add the tag.  Then, you can also edit or delete the tag.

. Click *Cluster* to view the overview page.
+
The tags are listed in the *Tags* section.  
. To add, edit, or delete the tags, click *Manage Tags*, and then perform the following:
+
[cols="25,75"]
|===

h| To perform this action...  h| Perform these steps...

a| Add a tag
a|
.. Click *Add Tag*.
.. Specify a key and its value or values (separate multiple values with commas).
.. Click *Save*.

a| Edit a tag
a|
.. Modify the content in the *Key* and *Values (optional)* fields.
.. Click *Save*.

a| Delete a tag
a| 
.. Click image:../media/icon_trash_can_white_bg.gif[trash can icon] next to the tag you want to delete.

|===
