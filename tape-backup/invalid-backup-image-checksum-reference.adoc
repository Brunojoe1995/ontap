= Invalid backup image checksum
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Invalid backup image checksum`

* *Cause*
+
The tape is corrupt.

* *Corrective action*
+
If the tape is corrupt, you cannot perform the restore operation.
