= No files were created
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`No files were created`

* *Cause*
+
A directory DAR was attempted without enabling the enhanced DAR functionality.

* *Corrective action*
+
Enable the enhanced DAR functionality and retry the DAR.
