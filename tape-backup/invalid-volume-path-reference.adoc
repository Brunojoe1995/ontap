= Invalid volume path
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Invalid volume path`

* *Cause*
+
The specified volume for the backup or restore operation is not found.

* *Corrective action*
+
Retry the job with a valid volume path and volume name.
