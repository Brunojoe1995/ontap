= Failed to read backup image
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Failed to read backup image`

* *Cause*
+
The tape is corrupt.

* *Corrective action*
+
If the tape is corrupt, you cannot perform the restore operation.
