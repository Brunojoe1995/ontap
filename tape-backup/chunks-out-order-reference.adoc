= Chunks out of order
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Chunks out of order`

* *Cause*
+
The backup tapes are not being restored in the correct sequence.

* *Corrective action*
+
Retry the restore operation and load the tapes in the correct sequence.
