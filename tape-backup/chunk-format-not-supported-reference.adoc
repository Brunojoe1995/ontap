= Chunk format not supported
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Chunk format not supported`

* *Cause*
+
The backup image is not of SMTape.

* *Corrective action*
+
If the backup image is not of SMTape, retry the operation with a tape that has the SMTape backup.
