= Tape write failed - new tape encountered media error
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Tape write failed - new tape encountered media error`

* *Cause*
+
The tape used for the backup is corrupted.

* *Corrective action*
+
Replace the tape and retry the backup.
