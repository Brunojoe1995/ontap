= Volume is temporarily in a transitional state
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Volume is temporarily in a transitional state`

* *Cause*
+
The volume being backed up is temporarily in an unmounted state.

* *Corrective action*
+
Wait for some time and perform the backup again.
