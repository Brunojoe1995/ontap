= ndmpd session session_ID not active
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`ndmpd session session_ID not active`

* *Cause*
+
The NDMP session might not exist.

* *Corrective action*
+
Use the `ndmpd status` command to view the active NDMP sessions.
