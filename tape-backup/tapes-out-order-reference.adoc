= Tapes out of order
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Tapes out of order`

* *Cause*
+
The first tape of the tape sequence for the restore operation does not have the image header.

* *Corrective action*
+
Load the tape with the image header and retry the job.
