= Mismatch in backup set ID
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Mismatch in backup set ID`

* *Cause*
+
The tape loaded during a tape change is not a part of the backup set.

* *Corrective action*
+
Load the correct tape and retry the job.
