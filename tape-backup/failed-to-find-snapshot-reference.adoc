= Failed to find snapshot
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Failed to find snapshot`

* *Cause*
+
The Snapshot copy specified for the backup is unavailable.

* *Corrective action*
+
Check if the specified Snapshot copy is available. If not, retry with the correct Snapshot copy.
