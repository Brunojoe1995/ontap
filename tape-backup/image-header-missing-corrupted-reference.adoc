= Image header missing or corrupted
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Image header missing or corrupted`

* *Cause*
+
The tape does not contain a valid SMTape backup.

* *Corrective action*
+
Retry with a tape containing a valid backup.
