= Tape write failed - new tape is already at the end of media
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Tape write failed - new tape is already at the end of media`

* *Cause*
+
There is not enough space on the tape to complete the backup.

* *Corrective action*
+
Replace the tape and retry the backup.
