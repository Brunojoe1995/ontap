= Destination qtree is read-only
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Destination qtree is read-only`

* *Cause*
+
The qtree to which the restore is attempted to is read-only.

* *Corrective action*
+
Try restoring the data to a different location.
