= Tape is currently in use by other operations
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Tape is currently in use by other operations`

* *Cause*
+
The tape drive is in use by another job.

* *Corrective action*
+
Retry the backup after the currently active job is finished.
