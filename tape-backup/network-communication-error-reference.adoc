= Network communication error
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Network communication error`

* *Cause*
+
Communication to a remote tape in an NDMP three-way connection has failed.

* *Corrective action*
+
Check the network connection to the remote mover.
