= Media error on tape write
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Media error on tape write`

* *Cause*
+
The tape used for the backup is corrupted.

* *Corrective action*
+
Replace the tape and retry the backup job.
