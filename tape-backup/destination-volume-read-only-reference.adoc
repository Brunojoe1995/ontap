= Destination volume is read-only
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Destination volume is read-only`

* *Cause*
+
The path to which the restore operation is attempted to is read-only.

* *Corrective action*
+
Try restoring the data to a different location.
