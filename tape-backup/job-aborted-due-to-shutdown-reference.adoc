= Job aborted due to shutdown
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Job aborted due to shutdown`

* *Cause*
+
The storage system is being rebooted.

* *Corrective action*
+
Retry the job after the storage system reboots.
