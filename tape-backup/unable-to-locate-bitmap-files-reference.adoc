= Unable to locate bitmap files
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Unable to locate bitmap files`

* *Cause*
+
The bitmap files required for the backup operation might have been deleted. In this case, the backup cannot be restarted.

* *Corrective action*
+
Perform the backup again.
