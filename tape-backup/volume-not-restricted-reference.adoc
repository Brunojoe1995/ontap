= Volume not restricted
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Volume not restricted`

* *Cause*
+
The destination volume to which data is being restored is not restricted.

* *Corrective action*
+
Restrict the volume and retry the restore operation.
