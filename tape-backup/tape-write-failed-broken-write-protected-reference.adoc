= Tape write failed - new tape is broken or write protected
:icons: font
:imagesdir: ../media/

[.lead]
* *Message*
+
`Tape write failed - new tape is broken or write protected`

* *Cause*
+
The tape used for the backup is corrupted or write-protected.

* *Corrective action*
+
Replace the tape and retry the backup.
